JLPT Vocabulary Quiz Site



Introduction

This was my first attempt at making a website.

Motivation behind this is that most Japanese quiz websites are very old fashioned ( as is with most Japanese websites ) so I figure I'd give it a shot to start off with something basic as my first JS project. It does what it sets out to do, which is test words that Jisho.org deems to be " N# test level ", but could use more features that I've yet implemented.

Features

JLPT N5, N4, N3, N2, N1 word banks can be individually enabled to be in the active bank

    ・These are gathered from reading the JSON of a google spreadsheet file that grabbed its info from jisho.org
      ➡➡➡NOTE: THIS IS SLOW ON LOADUP!! A much better alternative would be to have a server sided database instead of asking the client's side to load in data from a website...

    Wrong answers are given the correct answer in the Japanese phonetic system in case Chinese characters are used

      ・Done with also grabbing phonetics of words and inserting the word into the word bank array as a key, so it's very easy to compare user input to correct input
